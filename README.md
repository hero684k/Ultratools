## 🎯 Что это?

**UltraTools** — это лёгкий, быстрый и красивый набор повседневных мини-инструментов, работающий прямо в браузере. Никаких установок, регистраций и зависимостей. Просто открой страницу — и пользуйся.

Все данные сохраняются локально в твоём браузере (localStorage). Ничего не отправляется на сервер.

---

## ✨ Возможности

| Инструмент | Описание |
|-----------|----------|
| 🧮 **Калькулятор** | Поддержка клавиатуры (+, −, *, /), константы π, e, φ, √2, ln(2) |
| 📝 **Заметки** | Несколько вкладок с автосохранением в localStorage |
| ⏱️ **Таймер** | Обратный отсчёт с настраиваемым временем |
| ⏲️ **Секундомер** | Точный секундомер с миллисекундами |
| 🔐 **Генератор паролей** | Настройка длины, выбор символов (A-Z, a-z, 0-9, спецсимволы) |

## 🚀 Быстрый старт

### Способ 1 — GitHub Pages (рекомендуется)

1. Загрузи файлы в репозиторий
2. Перейди в **Settings → Pages**
3. Выбери ветку `main` и папку `/ (root)`
4. Нажми **Save**
5. Твой сайт будет доступен по адресу: `https://[твой-юзернейм].github.io/ultratools/`

### Способ 2 — Локальный запуск

1. Скачай репозиторий: `git clone https://github.com/hero684k/ultratools.git`
2. Открой `index.html` в любом браузере
3. Готово!
