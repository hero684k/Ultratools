## 🌐 Открыть сайт

👉 **[https://hero684k.github.io/Ultratools/](https://hero684k.github.io/Ultratools/)**

Просто открой ссылку в любом браузере. Ничего скачивать не нужно.

---

## ℹ️ Что это?

**UltraTools** — это набор простых и полезных инструментов, которые работают прямо в браузере. Никакой рекламы, регистраций и скрытых платежей. Всё бесплатно и навсегда.

Твои данные никуда не отправляются — всё хранится только в твоём браузере.

---

## 📖 Как пользоваться?

1. Открой сайт по ссылке выше
2. Выбери нужный инструмент в верхнем меню
3. Пользуйся!

Сайт работает на компьютерах, планшетах и телефонах.

---

## 🛠️ Инструменты

| Инструмент | Что делает |
|-----------|------------|
| 🧮 **Калькулятор** | Обычный калькулятор. Можно нажимать кнопки или печатать на клавиатуре. Есть кнопки для числа π, e и других констант |
| 📝 **Заметки** | Можно создавать несколько заметок, переключаться между ними. Всё автоматически сохраняется |
| ⏱️ **Таймер** | Обратный отсчёт. Поставь нужное время и запускай |
| ⏲️ **Секундомер** | Засекает время с точностью до миллисекунд |
| 🔐 **Генератор паролей** | Создаёт надёжные пароли. Можно выбрать длину и из каких символов делать |

## 📱 Как добавить на телефон

### Android (Chrome):
1. Открой сайт
2. Нажми на три точки вверху справа
3. Выбери **"Добавить на главный экран"**

### iPhone (Safari):
1. Открой сайт
2. Нажми на кнопку "Поделиться" (квадратик со стрелкой)
3. Выбери **"На экран «Домой»"**

Сайт откроется как обычное приложение.
